由于傻逼的中文让我屏幕一堆乱码
![[Pasted image 20260420200553.png]]
bad
![[Pasted image 20260420200727.png]]
解决方法

1. 重新生成 Locale 配置

  • 命令： sudo nano /etc/locale.gen

  • 操作： 确保 en_US.UTF-8 UTF-8 前面没有 #，而 zh_CN 开头的行全部加上 #。

  • 生效： 执行 sudo locale-gen --purge。

2. 强制系统环境为英文

  • 命令：
    sudo update-locale LANG=en_US.UTF-8 LANGUAGE=en_US.UTF-8 LC_ALL=en_US.UTF-8
  • 作用： 这会将系统全局变量永久改为英文，优先级最高。

3. 拦截 SSH 客户端的干扰（关键）

  • 命令： sudo nano /etc/ssh/sshd_config

  • 操作： 找到 AcceptEnv LANG LC_*，在行首添加 # 将其注释掉。

  • 生效： sudo systemctl restart ssh

  • 作用： 防止 Windows 客户端（GBK/GB18030 编码）强行把中文配置塞给树莓派。
